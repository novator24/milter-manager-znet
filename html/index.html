milter manager?

milter manager is a free software to protect you from spam mails and virus mails effectively with milter.

milter manager can be used with MTAs that supports milter like Sendmail and Postfix. It works on Ubuntu, CentOS, FreeBSD and so on.

Support

Questions and bug reports are accepted on mailing list. New release announce is also done on the mailing list. If you are using milter manager, it's a good idea that you subscribe the mailling list.

Development release

Packages of development releases for Ubuntu 8.04 LTS Hardy Heron

apt-line for: 

deb http://milter-manager.sourceforge.net/ubuntu/development/ hardy universe
deb-src http://milter-manager.sourceforge.net/ubuntu/development/ hardy universe

Packages are signed by key of milter manager. If you trust the key, you can register the key: 

% gpg --keyserver hkp://subkeys.pgp.net --recv-keys 1BD22CD1
% gpg --export 1BD22CD1 | sudo apt-key add -

Packages of development releases for CentOS 5.5

Packages are provided as yum repository. You need to register milter manager yum repository at the first time: 

% sudo rpm -Uvh http://milter-manager.sourceforge.net/centos/5/milter-manager-repository-1.0.0-1.noarch.rpm

You can install milter manager from the yum repository by the following command: 

% sudo yum install --enablerepo=milter-manager-development -y milter-manager

Join milter manager developement

milter manager uses git repository on GitHub. You can get the latest milter manager source code by the following command: 

% git clone https://github.com/milter-manager/milter-manager.git
If you want to send a patch for a new feature or a bug fix, please fork the milter manager repository and send a pull request. Patches are also accepted on mailing list.
